# Metaverse Weather System

A comprehensive weather and environment management plugin for **Unreal Engine**, built on top of the **Ultra Dynamic Sky** plugin.

This plugin provides a centralized system for controlling environmental conditions such as weather, lighting, atmosphere, and time of day through a clean and intuitive interface. It is designed to simplify the creation of immersive environments while remaining flexible enough for simulations, games, and visualization projects.

Rather than interacting directly with Ultra Dynamic Sky, the plugin provides a higher-level workflow for creating, editing, saving, and applying environmental conditions during development and at runtime.

> **Note:** This plugin requires **Ultra Dynamic Sky** to be installed in your Unreal Engine project.

---

## Features

* Dynamic weather control
* Rain, snow, hail, fog, dust, and haze support
* Multi-layer cloud configuration
* Wind configuration with multiple layers
* Day & night cycle management
* Manual and animated time control
* Environment preset creation and management
* Runtime weather updates
* Area-based environment customization
* Modern Slate-based editor interface
* XML-based pre-configured environement & weather presets

---

## Why This Plugin?

Managing weather and environmental settings directly can become difficult as projects grow in complexity. This plugin provides a single, organized workflow for handling environmental effects while keeping the experience simple for both developers and designers.

Whether you're creating an open-world game, a flight simulator, a training application, or an architectural visualization, the plugin offers an efficient way to manage dynamic environmental conditions.
